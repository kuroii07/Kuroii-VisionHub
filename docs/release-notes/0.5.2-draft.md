# Kuroii VisionHub v0.5.2 Release Notes Draft

Date: 2026-07-08

## Summary

`0.5.2` is a hotfix for a Windows release-build white-screen / heavy-lag issue observed after the Kuroii branding rebuild. DevTools showed that the DOM could be fully mounted while the native WebView2 window painted a blank white surface, so this release adds a Windows WebView2 GPU-compositor workaround.

## Fixes

- Added Windows WebView2 launch arguments: `--disable-gpu`, `--disable-gpu-compositing`, and `--disable-accelerated-2d-canvas`.
- Updated the boot HTML title and startup fallback copy to `Kuroii VisionHub`.
- Kept the Tauri identifier `studio.visionhub.app` unchanged for AppData/history/profile/credential compatibility.

## Validation checklist
