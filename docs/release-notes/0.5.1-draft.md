# Kuroii VisionHub v0.5.1 Release Notes Draft

Date: 2026-07-08

## Summary

`0.5.1` is the first Kuroii brand integration checkpoint for the former VisionHub Studio codebase. The product display name is now `Kuroii VisionHub` with the subtitle `AI Image Workflow Studio`.

## Brand integration

- Desktop and installer icons now use approved Kuroii app icon assets.
- Sidebar brand mark uses the Kuroii transparent app icon through the existing `src/ui/assets/visionhub-logo.png` compatibility path.
- Kuroii Cat expression and pose assets are copied into `src/ui/assets/kuroii/` for restrained future use in empty states, onboarding, success/help states, and update pages.
- Asset source paths, dimensions, and SHA256 values are recorded in `docs/kuroii-brand-asset-map.md`.

## Compatibility notes

- Internal npm/Cargo package name remains `visionhub-studio`.
- Tauri identifier remains `studio.visionhub.app` intentionally, so existing AppData, gallery/history, Provider profile ids, and system credential bindings remain available on this computer.
- Fresh installs on other computers will not automatically contain this machine's gallery/history/config unless the user manually migrates data.
- Provider request protocols, generation history structure, and credential secret ids are unchanged.
