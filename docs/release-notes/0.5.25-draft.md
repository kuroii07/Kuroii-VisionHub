## Compatibility and release notes

- This build keeps existing local data and credential compatibility identifiers unchanged.
- The Windows artifacts are unsigned; Windows SmartScreen may show an unknown-publisher warning.
- This release does not install, update, or remove AiMaMi, Clash/VPN, system proxy, DNS, gallery data, history, settings, caches, or credentials.
- MSI and NSIS installation, upgrade, and uninstallation QA remain part of the broader pre-v1.0 validation phase.
