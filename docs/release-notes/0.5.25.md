迁移到 `kuroii07/Kuroii-VisionHub` 新仓库后的首个预发布版本。

### 下载

- `Kuroii-VisionHub_0.5.25_x64-setup.exe`：推荐普通用户使用的当前用户安装包。

- 软件内“查看发布版本”和 README 下载入口已切换到新仓库。
- 完整保留 `main` 历史与参考图片分支。
- 保留 Tauri identifier、AppData、Provider profile id、系统凭据绑定和内部兼容标识，升级后不会主动清空历史记录、画廊、配置或凭据。
- 完成便携版、NSIS 与 MSI 构建及基础安装、启动、卸载验证。

## English

The first pre-release published after the repository migration to `kuroii07/Kuroii-VisionHub`.

### Downloads

- `Kuroii-VisionHub_0.5.25_x64-setup.exe`: recommended current-user installer for most users.
- `Kuroii-VisionHub.exe`: portable executable that does not require installation.
- `Kuroii-VisionHub_0.5.25_x64_en-US.msi`: MSI package for managed deployment; the default all-users path requires administrator privileges.
- `SHA256SUMS.txt`: SHA256 checksum list for all published artifacts.

### Highlights

- Moved the primary repository and default development remote to `kuroii07/Kuroii-VisionHub`.
- Updated the in-app release destination and README download links to the new repository.
- Preserved the complete `main` history and reference-image branch.
- Preserved the Tauri identifier, AppData, provider profile IDs, system credential bindings, and internal compatibility identifiers. Upgrading does not intentionally clear history, galleries, settings, or credentials.
- Built and validated the portable EXE, NSIS installer, and MSI package through basic installation, launch, and uninstallation checks.
- Unified verification passed with 133 frontend tests and 3 Rust tests. `npm audit` reported 0 high-severity vulnerabilities.

### Notes

- This is a pre-release. The Windows packages are not code-signed and may trigger an unknown-publisher or SmartScreen warning.
- The NSIS installer is recommended for most users. The MSI package requires administrator privileges for its default all-users installation path.
- Reinstalling on the same computer intentionally reuses existing AppData to preserve history, galleries, settings, and credential compatibility. A first installation on another computer does not inherit this data automatically.
- This release does not install, update, or remove AiMaMi, Clash / VPN, system proxy, DNS, gallery data, history, settings, caches, or credentials.
