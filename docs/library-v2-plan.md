# VisionHub Studio 作品画廊 V2 优化计划

## 目标

作品画廊 V2 的目标是从“生成历史列表”升级为“图片管理工具”。主界面优先展示图片，搜索、过滤、排序、详情和管理操作都以低占用、按需展开的方式提供。

## 核心原则

- 图片优先：默认视图最大化展示图片，不使用常驻双列或三列详情布局。
- 详情按需：图片详情侧栏默认隐藏，只在用户点击详情时打开。
- 控制收纳：搜索、过滤、视图、排序和新增入口集中到底部悬浮搜索栏。
- 元数据安全：收藏、标签、文件夹、备注等管理信息独立保存，不破坏现有生成历史。
- 渐进迭代：先保留现有数据结构，逐步增加图片管理能力。

## V2 第一阶段范围

### 主图区域

- 使用图片优先网格作为默认视图。
- 支持视图模式：
  - 瀑布流
  - 自适应网格
  - 正方形网格
  - 完整宽高比
  - 列表视图
- 图片卡片默认只展示必要状态：
  - 缩略图
  - 收藏按钮
  - 成功 / 失败状态
  - 图生图参考标记
  - Provider / 模型信息按显示设置决定是否出现

### 详情侧栏

- 默认隐藏。
- 点击图片详情按钮后从右侧滑出。
- 关闭后主图区域恢复完整宽度。
- 详情内容：
  - 大图预览
  - Prompt
  - Provider
  - 模型
  - 尺寸、质量、数量
  - 生成耗时
  - 创建时间
  - 本地文件路径
  - 参考图来源
  - 原始 raw / 错误信息入口
  - 操作按钮：复制 Prompt、打开文件、设为参考图、删除记录

### 底部悬浮搜索栏

- 放在作品画廊页面底部居中。
- 左侧是菜单按钮。
- 中间是搜索输入框。
- 右侧是加号按钮。
- 页面底部预留安全空间，避免搜索栏遮挡最后一排图片。

搜索范围：

- Prompt
- Provider
- 模型
- 标签
- 收藏状态
- 文件夹 / 收藏集
- 文生图 / 图生图
- 成功 / 失败
- 日期范围
- 本地文件名 / 路径
- 错误信息

### 左侧菜单按钮

菜单项：

- 隐藏搜索栏
- 显示过滤器
- 网格样式
- 显示设置
- 排序方式
- 缩略图放大
- 缩略图缩小

网格样式：

- 瀑布流
- 自适应
- 正方形
- 完整宽高比
- 列表视图

显示设置建议：

- 显示 / 隐藏 Prompt 摘要
- 显示 / 隐藏 Provider 徽标
- 显示 / 隐藏模型名
- 显示 / 隐藏失败记录
- 显示 / 隐藏图生图参考标记
- 显示 / 隐藏本地保存状态
- 紧凑间距 / 舒适间距

排序方式建议：

- 最新优先
- 最早优先
- 收藏优先
- Provider 分组
- 模型分组
- 生成耗时
- 图片尺寸
- 文件名
- 最近查看
- 最近使用为参考图

### 右侧加号按钮

建议功能：

- 新建文件夹
- 新建收藏集
- 导入本地图片
- 从剪贴板导入
- 批量导入文件夹
- 新建标签
- 从失败记录恢复图片

第一阶段可以先做菜单入口和占位提示，后续逐项接真实功能。

## 收藏与元数据

收藏功能不直接写入生成历史记录，建议新增独立元数据文件。

建议文件：

```text
library-meta.json
```

建议按 `recordId` 或本地图片路径保存：

- favorite: 是否收藏
- tags: 标签
- folderId: 所属文件夹
- collectionIds: 收藏集
- note: 自定义备注
- rating: 手动评分
- lastViewedAt: 最近查看时间
- lastUsedAsReferenceAt: 最近作为参考图使用时间

这样可以避免生成历史结构变化时丢失人工整理信息。

## 记录健康状态

后续可为每条记录计算健康状态：

- 图片存在
- 图片缺失
- raw 可恢复
- 仅失败记录
- 本地路径失效
- 文件存在但历史缺失

健康状态可用于：

- 一键修复
- 重新索引图库目录
- 从 raw 恢复图片
- 清理无效记录

## 分阶段实施

### 阶段 1：画廊 V2 基础体验

- 保留现有生成历史数据结构。
- 加入底部悬浮搜索栏。
- 加入菜单按钮和视图样式状态。
- 加入收藏按钮和收藏筛选。
- 详情改为按需打开的右侧抽屉。

### 阶段 2：元数据和管理能力

- 新增 `library-meta.json`。
- 实现收藏、标签、文件夹、收藏集。
- 支持导入本地图片和批量导入文件夹。
- 增加显示设置和排序方式持久化。

### 阶段 3：文件健康与修复

- 计算记录健康状态。
- 支持 raw 恢复图片。
- 支持重新索引图库目录。
- 支持失效记录清理。

### 阶段 4：高级图片管理

- 多选和批量操作。
- 右键菜单。
- 拖拽移动到文件夹 / 收藏集。
- 批量打标签。
- 批量复制 Prompt / 路径。

## 暂不做

- 不在第一阶段重做整个历史数据结构。
- 不默认删除磁盘图片。
- 不默认引入复杂数据库。
- 不把详情侧栏做成常驻布局。
- 不让底部搜索栏承载过多筛选项，复杂筛选放到菜单弹层。

## 验收标准

- 默认进入画廊时，主视觉是图片网格。
- 不打开详情时，页面没有常驻右侧详情栏。
- 搜索栏位于底部居中，不遮挡最后一排图片。
- 菜单按钮能收纳过滤、视图、显示设置、排序和缩放入口。
- 收藏状态可见、可切换、可搜索或过滤。
- 详情抽屉打开和关闭不破坏主网格的浏览体验。
